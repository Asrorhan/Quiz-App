Simple Quiz App

This is a small Quiz App I built while practicing JavaScript.
It asks multiple-choice questions and shows the score at the end.

I made this project to learn DOM manipulation, events, and basic logic in JavaScript.

Features

Multiple questions

Shows correct/incorrect answers

Next button to move to the next question

Score result at the end

Restart quiz option

Clean and simple UI

Technologies

HTML

CSS

JavaScript (Vanilla JS)

What I Learned

How to create elements using JavaScript

How to update text dynamically

How to loop through questions

How to handle button click events

How to disable and style buttons

How to Run

Just download the project and open index.html in your browser.
No installation needed.
